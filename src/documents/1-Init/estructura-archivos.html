Estructura de Archivos

Para una mejor semántica se propone separar los archivos base (src) de los procesados. Los archivos finales procesados para desarrollo se crearán en un directorio llamado (dev) mientras que los optimizado listos para producción se crearan en un derectorio llamado (prod).
