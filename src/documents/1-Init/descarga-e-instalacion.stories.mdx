# Descarga:

En la siguiente ruta encontrarás el repositorio frontend I2b https://github.com/I2BLab/front-component 

Los ** *.scss, *.pug y *.js ** son el código fuente de nuestros ** CSS, html y JavaScript. ** El directorio ** dev ** contiene los archivos css, html, js compilados.

A continuación la url para clonar el repositorio con git: https://github.com/I2BLab/front-component.git

# Instalación:

- ** - Paso 1: ** Abra la terminal de comandos.
- ** - Paso 2: ** Diríjase a ** /front-component/docker. **
- ** - Paso 3: ** Ejecute la siguiente línea de comando para construir los contenedores y levantar el sistema: ** sh app.sh up **

# Branch:

Actualmente contamos con dos Branch
- ** master: ** En este branch se encuentra los archivos bases de los componentes listos para ser usados en cualquier proyecto.
- ** documentation: ** Este branch es utilizado para crear y actualizar la documentación de los componentes.

# Documentación de componentes:

Después de descargar e instalar puede ver la documentación de forma local ingresando a: ** http://localhost:65200 ** ubicado en el branch ** documentation **

# Repositorio de componentes:
En el branch ** master ** encontrará el repositorio componente, después de iniciar el contenedor puede ver los componentes frontend en la siguiente ruta: ** http://localhost:3000 **
