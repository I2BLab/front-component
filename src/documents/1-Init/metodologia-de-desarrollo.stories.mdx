# Metodología de desarrollo

<Meta title="Estándar de desarrollo Front-end/Metodología de desarrollo" />

1. Recibir la maqueta por parte del diseñador la cual debe estar desarrollada en base al repositorio de componentes.
2. Realizar cambios en el archivo de configuración src/scss/_config.scss según lo entregado por el diseñador de la maqueta.
3. Iniciar maquetación basado en el repositorio de componentes.
4. Tener en cuenta que cada proyecto puede tener componentes en específico que no estén contemplado en el repositorio para estos caso se debe evaluar si se crea un nuevo componente y se agrega el repositorio o solo se mantiene en el proyecto en ejecución.
