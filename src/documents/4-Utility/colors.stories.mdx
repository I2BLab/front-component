# Colores

<Meta title="Utilidades/Colores" />

Si deseea trasmitir por medio del color algún significado, con las siguientes clases de la utilidad de color puede realizarlo.

## Ejemplo

<Preview>
  <Story
    name="Basic"
    height="150px"
    parameters={{ docs: { transformSource: (src) => `transformed: ${src}` } }}
  >
    {`<p class="text-default">.text-primary</p>
        <p class="text-secondary">.text-secondary</p>
        <p class="text-danger">.text-danger</p>
        <p class="text-black">.text-black</p>
        <p class="text-muted bg-blakc">.text-muted</p>
        <p class="text-gray-100 bg-black">.text-gray-100</p>
        <p class="text-white bg-black">.text-white</p>
        <p class="text-red-400">.text-red-400</p>`}
  </Story>
</Preview>

## Enlaces

Las clases de texto contextual también funcionan bien en los enlaces. Tenga en cuenta que la clase `.text-whitey` `.text-muted` no tiene un estilo de enlace adicional más allá del subrayado.

<Preview>
  <Story
    name="Enlaces"
    height="150px"
    parameters={{ docs: { transformSource: (src) => `transformed: ${src}` } }}
  >
    {`  <p><a href="#" class="text-default">.text-primary</a></p>
        <p><a href="#" class="text-secondary">.text-secondary</a></p>
        <p><a href="#" class="text-danger">.text-danger</a></p>
        <p><a href="#" class="text-black">.text-black</a></p>
        <p><a href="#" class="text-muted bg-blakc">.text-muted</a></p>
        <p><a href="#" class="text-gray-100 bg-black">.text-gray-100</a></p>
        <p><a href="#" class="text-white bg-black">.text-white</a></p>
        <p><a href="#" class="text-red-400">.text-red-400</a></p>`}
  </Story>
</Preview>

## Background

Defina fácilmente el fondo de un elemento en cualquier clase contextual. La clase `.bg-color` permite definir color de fondo a su elemento. 

<Preview>
  <Story
    name="Background"
    height="350px"
    parameters={{ docs: { transformSource: (src) => `transformed: ${src}` } }}
  >
    {`  <div class="p-10 mb-10 bg-default text-white">.bg-default</div>
        <div class="p-10 mb-10 bg-secondary text-white">.bg-secondary</div>
        <div class="p-10 mb-10 bg-danger text-white">.bg-danger</div>
        <div class="p-10 mb-10 bg-black text-white">.bg-black</div>
        <div class="p-10 mb-10 bg-muted text-white">.bg-muted</div>
        <div class="p-10 mb-10 bg-gray-100 text-black">.bg-gray-100</div>
        <div class="p-10 mb-10 bg-red-400 text-white">.bg-red-400</div>
     `}
  </Story>
</Preview>
