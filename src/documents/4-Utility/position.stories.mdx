# Position

<Meta title="Utilidades/Position" />

Utilice estas utilidades abreviadas para configurar rápidamente la posición de un elemento.

## Fixed top

Para Colocar un elemento en la parte superior del contenedor. Asegúrese de usar la clase `fixed-top`

~~~
<div class="fixed-top">...</div>
~~~


## Fixed bottom

Para Colocar un elemento en la parte inferior del contenedor. Asegúrese de usar la clase `fixed-bottom`

~~~
<div class="fixed-bottom">...</div>
~~~


## Sticky top


Para Colocar un elemento en la parte superior del contenedor, pero solo después de desplazarse. Asegúrese de usar la clase `.sticky-top`

~~~
<div class="sticky-top">...</div>
~~~
