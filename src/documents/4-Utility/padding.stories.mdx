# Padding

Esta utilidad permite dar padding a los diferentes elementos del Html. 

`[ Propiedad ][ Dirección ]-[ Tamaño ]`

**Propiedad:** La propiedad aplica el tipo de espaciado.

**Dirección:** Designa el lado al que se aplica la propiedad.

- **t-** aplica el espacio para padding-top
- **b-** aplica el espacio para padding-bottom
- **l-** aplica el espacio para padding-left
- **r-** aplica el espacio para padding-right
- **x-** aplica el espacio para ambos *-left y *-right
- **y-** aplica el espacio para ambos *-top y *-bottom

**Tamaño:** Corresponde al espaciado que deseas asignar definido en el archivo **/src/scss/_config.scss**

## Padding m

Como podras ver en el siguiente ejemplo agregamos padding general de 10px.

~~~
<div class="m-10">
    <div>
        Esto es codigo de prueba
    </div>
</div>
~~~

## Padding mt, mr, mb, ml

Como podras ver en el siguiente ejemplo agregamos:
- padding-top: 10
- padding-right: 10
- padding-bottom: 10
- padding-left: 10

~~~
<div class="mt-10 mr-10 mb-10 ml-10">
    <div>
        Esto es codigo de prueba
    </div>
</div>
~~~

## Padding mx, my

Como podras ver en el siguiente ejemplo agregamos un padding horizontal de 10px y un padding vertical de 20px

~~~
<div class="mx-10 my-20">
    <div>
        Esto es codigo de prueba
    </div>
</div>
~~~

## Padding @media

También podemos personalizar las utilidades flexibles para aplicarlas en función de varios puntos de interrupción.

- **sm**: 640px
- **md**: 768px
- **lg**: 1024px
- **xl**: 1280px

~~~
<div class="sm:mx-10 md:my-20 lg:mt-10 xl:m-10">
    <div>
        Esto es codigo de prueba
    </div>
</div>
~~~
