# Flex

Controla el diseño de los contenedores flexibles con alineación, justificación y más con utilidades flex

Usando utilidades de **d-flex**, podemos convertir cualquier elemento en un contenedor flexbox transformando elementos secundarios directos en elementos flexibles.

Como podras ver en el siguiente ejemplo agregamos Flex.

<Preview>
  <Story name="Flex" height="50px">
    {`<div class="d-flex">
        <div>
            I'm a flexbox container!
        </div>
    </div>`}
	</Story>
</Preview>

También podemos personalizar las utilidades flexibles para aplicarlas en función de varios puntos de interrupción.

- **sm**: 640px **=>** sm:d-flex
- **md**: 768px **=>** sm:d-flex
- **lg**: 1024px **=>** sm:d-flex
- **xl**: 1280px **=>** sm:d-flex

~~~
<div class="sm:d-flex md:d-flex lg:d-flex xl:d-flex">
    <div>
        Esto es codigo de prueba
    </div>
</div>
~~~

## Flex Direction Row y Reverse

Por defecto, d-flex se aplica flex-direction: row y generalmente se puede omitir. Sin embargo, puede haber situaciones en las que necesite definirlo explícitamente.

<Preview>
    <Story name="Flex Direction Row" height="100px">
        {`<div class="d-flex flex-row">
            <div class="bd p-10">
                Flex item 1
            </div>
            <div class="bd p-10">
                Flex item 2
            </div>
            <div class="bd p-10">
                Flex item 3
            </div>
        </div>
        <div class="d-flex flex-row-reverse">
            <div class="bd p-10">
                Flex item 1
            </div>
            <div class="bd p-10">
                Flex item 2
            </div>
            <div class="bd p-10">
                Flex item 3
            </div>
        </div>`}
    </Story>
</Preview>

## Flex Direction Column y Reverse

Las clases de utilidad flex-column y flex-column-reverse se pueden utilizar para cambiar la orientación del contenedor flexbox.

<Preview>
    <Story name="Flex Direction Column" height="300px">
        {`<div class="d-flex flex-column mb-30">
            <div class="bd p-10">
                Flex item 1
            </div>
            <div class="bd p-10">
                Flex item 2
            </div>
            <div class="bd p-10">
                Flex item 3
            </div>
        </div>
        <div class="d-flex flex-column-reverse">
            <div class="bd p-10">
                Flex item 1
            </div>
            <div class="bd p-10">
                Flex item 2
            </div>
            <div class="bd p-10">
                Flex item 3
            </div>
        </div>`}
    </Story>
</Preview>
