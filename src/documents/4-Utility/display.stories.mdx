# Display

permiten controlar la visualización del contenido. 
Esto incluye ser visible condicionalmente según la ventana 
gráfica actual o el tipo de visualización del elemento real.

**{breakpoint:}** **-** **{d}** **-** **{value}**

- **sm**: 640px **=>** sm:d-{value}
- **md**: 768px **=>** md:d-{value}
- **lg**: 1024px **=>** lg:d-{value}
- **xl**: 1280px **=>** xl:d-{value}

## Propiedades

- **d-none**
- **d-inline**
- **d-inline-block**
- **d-block**
- **d-table**
- **d-table-row**
- **d-table-cell**
- **d-flex**
- **d-inline-flex**
