Contenedor

Los contenedores son el elemento de diseño más básico y son necesarios cuando se utiliza nuestro sistema de cuadrícula predeterminado. Los contenedores se usan para contener, rellenar y (a veces) centrar el contenido dentro de ellos. Si bien los contenedores se pueden anidar, la mayoría de los diseños no requieren un contenedor anidado.

Viene contenedores diferentes:

Extra Pequeña < 640px Pequeña ≥ 640px Mediana ≥ 768px Grande ≥ 1024px Extra Grande ≥ 1280px

Basico

Clase por defecto .container es un contenedor que su max-width cambia en cada breakpoint.

.container

Fluido

Use .container-fluid para un contenedor de ancho completo, que abarca todo el ancho de la ventana gráfica.

.container-fluid

Responsivo

Le permiten especificar una clase que es 100% de ancho hasta que se alcanza el breakpoint especificado, después de lo cual aplicamos max-widths para cada uno de los breakpoints. Por ejemplo, .container-sm tiene un ancho del 100% para comenzar hasta que se alcanza el breakpoint sm, donde se ampliará con md, lg y xl.

Responsivo breakpoints

Todo el desarrollo de contenedores esta basado en móvil primero. Estos breakpoints se basan principalmente en los anchos mínimos de la vista y nos permiten escalar elementos a medida que cambia la vista.

@include media-breakpoint-up(sm) { ... }
@include media-breakpoint-up(md) { ... }
@include media-breakpoint-up(lg) { ... }
@include media-breakpoint-up(xl) { ... }

.custom-class {
    display: none;
}
@include media-breakpoint-up(sm) {
    .custom-class {
        display: block;
    }
}

Ocasionalmente utilizamos consultas de medios que van en la otra dirección (el tamaño de pantalla dado max a menos):

@include media-breakpoint-down(xs) { ... }
@include media-breakpoint-down(sm) { ... }
@include media-breakpoint-down(md) { ... }
@include media-breakpoint-down(lg) { ... }

También hay consultas de medios y mixins para apuntar a un solo segmento de tamaños de pantalla utilizando los anchos de breakpoint mínimo y máximo:

@include media-breakpoint-only(xs) { ... }
@include media-breakpoint-only(sm) { ... }
@include media-breakpoint-only(md) { ... }
@include media-breakpoint-only(lg) { ... }
@include media-breakpoint-only(xl) { ... }

Del mismo modo, las consultas de medios pueden abarcar varios anchos de breakpoint:

@include media-breakpoint-between(md, lg) { ... }
