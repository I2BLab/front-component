# Grid

## Breackpoints

Para las grids que son iguales desde el dispositivo más pequeño hasta el más grande, use las clases .col y .*:col. Especifique una clase numerada cuando necesite una columna de tamaño particular; de lo contrario, siéntase libre de apegarse a .col.
