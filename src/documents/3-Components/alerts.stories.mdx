## Alerts

Mensajes contextuales con acciones típicas en una aplicación.
Los mensajes de alerta están disponibles para cualquier longitud de texto y contienen un botón de cerrar. 

Los alerts se crean con la clase **alert** seguida de una clase de contexto. Las clases de contexto son:

1. **alert--success:** Alerta de éxito.
2. **alert--warning:** Alerta de advertencia.
3. **alert--danger:** Alerta de error.
4. **alert--info:** Alerta de información.

### Alert success

Indica una acción exitosa o positiva.

<Preview>
    <Story name="Alert success" height="50px">
		{
            `<div class="alert alert--success mt-10">
            <a class="alert__btn-close" href="javascript:;" onclick="this.parentElement.style.display='none'">×</a>
            <span>Success Lorem ipsum dolor sit amet.</span>
            </div>`
        }
	</Story>
</Preview>
