# Collapse

<Meta title="Componentes/Collapse" />

Una Card es un contenedor de contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido, colores de fondo contextuales y  opciones de visualización.

## Ejemplo

A continuación se muestra un ejemplo de una tarjeta básica con contenido mixto y un ancho fijo. Las tarjetas no tienen un ancho fijo para comenzar, por lo que, naturalmente, ocuparán todo el ancho de su elemento padre. Esto se personaliza fácilmente con la diversas  opciones de tamaño
