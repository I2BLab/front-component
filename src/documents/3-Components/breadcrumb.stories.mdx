# Breadcrumb

<Meta title="Componentes/Breadcrumb" />

Inidica la ubicación de la página actual dentro de una jerarquía de navegación que agrega separadores automáticamente.

## Ejemplo


<Preview>
  <Story
    name="Basic"
    height="100px"
    parameters={{ docs: { transformSource: (src) => `transformed: ${src}` } }}
  >
    {`<div class="mt-20"> 
        <ol class="breadcrumb mt-10">
            <li class="breadcrumb__item">Home</li>
        </ol>
        <ol class="breadcrumb mt-10">
            <li class="breadcrumb__item">
                <a href="#">Home</a>
            </li>
            <li class="breadcrumb__item">Página Actual</li>
        </ol>
        <ol class="breadcrumb mt-10">
            <li class="breadcrumb__item"><a href="#">Home</a></li>
            <li class="breadcrumb__item"><a href="#">Opción</a></li>
            <li class="breadcrumb__item">Página Actual</li>
        </ol>
      </div>`}
  </Story>
</Preview>

###  Cambiar el separador

Los separadores se agregan automáticamente en CSS a través de ::before y content. Se pueden cambiar cambiando .breadcrumb__item.
