Panels

Para encerrar cierto contenido dentro de una caja puedes hacer uso de un panel un ejemplo sencillo es aplicar la clase .panel a un elemento div. 
                
    <div class="panel">
    <span>Este es un panel.</span>
    </div>
Puedes agregar modificadores para cambiar el color de fondo, los margenes internos utilizando por ejemplos las clases .bg-default, .text-white .p-10 puedes modificar el color de fondo, el color del texto y el padding del panel. 
                
    <div class="panel bg-default text-white p-10">
        <span>Este es otro panel</span>
    </div>
