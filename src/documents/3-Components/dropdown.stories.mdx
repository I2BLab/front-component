# Dropdown

Los menús desplegables son superposiciones contextuales conmutables para mostrar listas de enlaces y más. Se hacen interactivos con el complemento de JavaScript.

## Normal

<Preview>
  <Story
		name="Normal"
		height="80px"
	>
		{`<button class="btn btn-menu bg-default text-white" data-js="dropdown" data-open="#menu1">Dropdown Normal</button>
			<div>
				<div class="nav nav-menu nav-vert bg-black text-white" id="menu1">
					<a class="nav-item nav--button active" href="javascript:;">Home</a>
					<a class="nav-item nav--button" href="#">Link 1</a>
					<a class="nav-item nav--button" href="#">Link 2</a>
					<a class="nav-item nav--button" href="#">Link 3</a>
				</div>
			</div>
			<script>
				import iDropdown from '../../js/components/i-dropdown';
				$(function () {
					$js('dropdown').iDropdown();
				});
			</script>`}
	</Story>
</Preview>
