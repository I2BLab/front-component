## Progress bar

Los progress bar se crean con la clase **bg-muted** la cual será la clase de la capa padre dentro debe contener
un elemento con un width el cual puede ser cambiado con algo de javascript para darle movimiento. 

~~~
<div class="bg-muted mt-10">
    <div class="bg-default text-center p-5" style="width:25%">25%</div>
</div>
~~~
