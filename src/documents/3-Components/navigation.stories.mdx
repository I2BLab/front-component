## Navigation

Es un elemento que nos permite navegar entre las diferentes páginas de una aplicación web.
El navigation esta compuesto por una capa contenedora con la clase **nav**, dentro contiene 
elementos hijos con las clases  **nav__item nav--button** y la clase **nav__item--active** que determina el elemento activo en 
el navigation.

A continuación 2 ejemplos de Navigation.
