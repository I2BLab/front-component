# Tablas

<Meta title="Componentes/Table" />

## Basic
Las tablas de datos muestran conjuntos de datos en filas y columnas.

### Ejemplo

Para el uso generalizado de tablas simplemente agregue la clase base `.tablea` cualquiera `<table>`, luego extiéndala con estilos personalizados o nuestras diversas clases de modificadores incluidos.


<Preview>
  <Story
    name="Basic"
    height="120px"
    parameters={{ docs: { transformSource: (src) => `transformed: ${src}` } }}
  >
    {`<table class="table">
            <thead>
                <tr>
                    <th>Nombre</th>
                    <th>Apellido</th>
                    <th>Edad</th>
                </tr>
            </thead>
            <tbody>
                <tr>
                    <td>Bruno</td>
                    <td>Diaz</td>
                    <td>35</td>
                </tr>
                <tr>
                    <td>Clark</td>
                    <td>Ken</td>
                    <td>25</td>
                </tr>
            </tbody>
        </table>`}
  </Story>
</Preview>

## Tabla con header color

Para el uso generalizado de tablas simplemente agregue la clase base `.table-all` a cualquier `<table>`, luego para agregar el color al titulo de la tabla asigne la clase `.bg-default` para definir el color por default, el background puede variar teniendo en cuenta el estandar de colores deinido.

### Ejemplo

<Preview>
  <Story
    name="Header color"
    height="120px"
    parameters={{ docs: { transformSource: (src) => `transformed: ${src}` } }}
  >
    {`<table class="table-all">

                    <th>Edad</th>
                </tr>

                <tr>
                    <td>Clark</td>
                    <td>Ken</td>
