Breadcrumb

Inidica la ubicación de la página actual dentro de una jerarquía de navegación que agrega separadores automáticamente.

Ejemplo

  1. Home
  1. Home
  2. Página Actual
  1. Home
  2. Opción
  3. Página Actual
            
                <ol class="breadcrumb mt-10"><li class="breadcrumb__item">Home</li></ol><ol class="breadcrumb mt-10"><li class="breadcrumb__item"><a href="#">Home</a></li><li class="breadcrumb__item">Página Actual</li></ol><ol class="breadcrumb mt-10"><li class="breadcrumb__item"><a href="#">Home</a></li><li class="breadcrumb__item"><a href="#">Opción</a></li><li class="breadcrumb__item">Página Actual</li></ol>

Cambiar el separador

Los separadores se agregan automáticamente en CSS a través de ::before y content. Se pueden cambiar cambiando .breadcrumb__item.
