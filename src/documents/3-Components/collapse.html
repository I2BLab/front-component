Collapse

Cambie la visibilidad del contenido con algunas clases y nuestros complementos de JavaScript.

Como funciona

El complemento JavaScript contraer se utiliza para mostrar y ocultar contenido. Los botones o anclas se usan como disparadores que se asignan a elementos específicos que se alternan.

Ejemplo Objetivos múltiples

A presionar los elementos buttons, puede mostrar y ocultar múltiples elementos al hacer referencia a ellos con un selector JQuery en su atributo href o data-target.

Haga clic en los botones de abajo para mostrar y ocultar otro elemento.
