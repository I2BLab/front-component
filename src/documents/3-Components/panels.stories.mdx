## Panels

Para encerrar cierto contenido dentro de una caja puedes hacer uso de un panel 
un ejemplo sencillo es aplicar la clase __.panel__ a un elemento div.	

~~~
<div class="panel">
  <span>Este es un panel.<span/>
</div>
~~~
