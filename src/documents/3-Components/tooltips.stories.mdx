# Tooltips

<Meta title="Componentes/Tooltips" />

Los tooltips son sugerencias textuales pequeñas e interactivas para elementos principalmente gráficos. Al usar íconos para acciones, puede usar una información sobre herramientas para aclarar a las personas sobre su función.

## Ejemplo

Desplácese sobre los enlaces a continuación para ver información sobre los elementos:

Para visualizar las su textuales simplemente agregue la clase `tooltip__text tooltip__text--right` a cualquier contanedor con la clase `.tooltip`.

Las clases puede variar teniendo en cuenta la ubicación donde quiere que se visualice la sugerencia textual.

- **Default**: `tooltip__text`
- **Left**: `tooltip__text tooltip__text--left`
- **Right**: `tooltip__text tooltip__text--right`
- **Top**: `tooltip__text tooltip__text--top`
- **Bottom**: `tooltip__text tooltip__text--bottom`
- **Animated**: `tooltip__text tooltip__text--ani`
