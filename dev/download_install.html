Descarga e instalación

En la siguiente ruta encontrarás el repositorio front-end https://github.com/I2BLab/front-component

Estructura de archivos

Con la siguiente url podrás clonar el repositorio de componentes front-end https://github.com/I2BLab/front-component.git

Los *.scss, *.pug y *.js son el código fuente de nuestros CSS, html y JavaScript. El directorio dev contiene los archivos css, html, js compilados. 

Pasos para la instalación:

  • Paso 1: Abra la terminal de comandos.
  • Paso 2: Diríjase a /front-component/docker.
  • Paso 3: Ejecute la siguiente línea de comando para construir los contenedores sh app.sh build.

Ejecución:

Para ejecutar los componente siga los siguientes pasos:

  • Paso 1: Diríjase a /front-component/docker
  • Paso 2: Ejecute la siguiente línea de comando sh app.sh up
  • Paso 3: Ingrese la siguiente url en el navegador http://localhost:3000/
